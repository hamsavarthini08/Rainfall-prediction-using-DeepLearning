About The Project

This project utilizes state-of-the-art deep learning techniques including LSTM and CNN-LSTM networks to predict rainfall patterns across Tamilnadu. By analyzing historical weather data including temperature, humidity, pressure, wind patterns, and geographical coordinates, our system provides accurate location-based rainfall predictions.
