About The Project

Advanced Deep Learning Based Rainfall Prediction System for Tamilnadu

Project Objective

The primary objective of this project is to develop an accurate, location-based rainfall prediction system for Tamilnadu using advanced deep learning techniques. By analyzing historical weather patterns and real-time data, we aim to provide reliable predictions that can aid in agricultural planning, disaster management, and daily decision-making.

Deep Learning Approach

We employ two state-of-the-art deep learning architectures: LSTM (Long Short-Term Memory) and CNN-LSTM hybrid networks. These models are specifically designed to capture temporal dependencies in weather data and spatial patterns across different geographical locations in Tamilnadu.

Model Architecture

LSTM Network

  • 3 LSTM layers with dropout
  • Sequence length: 30 days
  • 100-50-25 units
  • Adam optimizer
  • MSE loss function

CNN-LSTM Hybrid

  • 2 Conv1D layers (64,32 filters)
  • MaxPooling1D layers
  • 2 LSTM layers (50,25 units)
  • Dropout for regularization
  • Dense output layer
