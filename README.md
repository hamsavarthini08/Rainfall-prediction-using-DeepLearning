# Rainfall Prediction System for Tamilnadu

## Overview
Advanced deep learning-based rainfall prediction system for Tamilnadu using LSTM and CNN-LSTM networks.

## Features
- Location-based rainfall prediction
- Past, present, and future weather data
- Wind direction prediction
- Interactive dashboard with charts
- Weekly forecast generation
- Responsive modern UI

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript, Chart.js
- **Backend**: Python, Flask
- **Deep Learning**: TensorFlow, LSTM, CNN-LSTM
- **Data Processing**: Pandas, NumPy, Scikit-learn
